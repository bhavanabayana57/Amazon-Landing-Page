# 🛒 Amazon Landing Page

A responsive Amazon-inspired landing page built to showcase modern front-end web development concepts. The project provides an attractive shopping interface with interactive features for a smooth user experience.

## 🚀 Live Demo

🔗 https://bhavanabayana57.github.io/Amazon-Landing-Page/

## ✨ Features

- Responsive Design
- Product Search
- Product Category Filters
- Add to Cart / Remove from Cart
- Shopping Cart Counter
- Dark Mode Toggle
- Hero Section with Auto Slider
- Smooth Navigation
- About Us Section
- Contact Us Section
- Back to Top Button
- Toast Notifications

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 Project Objective

The objective of this project is to create a modern and responsive e-commerce landing page that demonstrates front-end development skills, interactive user interface design, and responsive web design principles.
